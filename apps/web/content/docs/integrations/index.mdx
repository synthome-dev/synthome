# Integrations

Synthome can integrate with external services for storage, delivery, and more.

## Available Integrations

| Integration                           | Use Case                        |
| ------------------------------------- | ------------------------------- |
| [CDN Storage](/docs/integrations/cdn) | Store and serve generated media |

## Storage

Generated media can be stored in cloud storage services:

- **Amazon S3**: AWS object storage
- **Cloudflare R2**: S3-compatible storage with no egress fees

See [CDN Storage](/docs/integrations/cdn) for setup instructions.
